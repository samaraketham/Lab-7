Middleware Setup:
express.urlencoded: Middleware to parse incoming request bodies with urlencoded payloads. This middleware is used to parse form data sent via POST requests (app.use(express.urlencoded({ extended: true }));).
express.static: Middleware to serve static files from the 'public' directory. For example, it can serve an HTML file named 'lab 7.html' located in the 'public' directory (app.use(express.static('public'));).
